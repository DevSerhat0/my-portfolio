## 📖 My Learning Journey
I am a 20-year-old, self-taught Junior Front-End Developer candidate. I believe that watching tutorial videos is not enough to learn programming; real learning happens when you write the code yourself, break things, and fix them.

I built this project to solidify my understanding of modern web development. Instead of just writing code that "works," I focused on writing code that makes sense. During the development process, I collaborated with an AI mentor (acting as my Senior Developer) to help me understand the core logic of React. Together, we:

- **Refactored** the initial messy code into a clean, modular component architecture (`ProfileImage`, `Content`, `SocialMedia`).
- Separated the raw content into a `Data.js` file to simulate how real-world applications fetch data.
- Migrated from global CSS to **CSS Modules** to understand component-level styling and prevent class name collisions.

This project might look like a simple card on the outside, but behind the scenes, it represents a huge leap in my understanding of clean code principles and React fundamentals!
